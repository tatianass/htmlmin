## Prerequisites

- `just`: [see here](https://github.com/cajsey/just)
- `python3` (>=3.13)
- `uv` [see here](https://github.com/astral-sh/uv)
- `pre-commit`

Activate the virtual environment with :
```bash
source .venv/bin/activate
```

Documentation: https://htmlmin.readthedocs.io/en/latest/
